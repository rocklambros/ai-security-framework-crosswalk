\section{Ablation Study}
\label{sec:ablations}

We conduct ablation experiments to quantify the contribution of each
feature family. All ablations use the same LightGBM hyperparameters
(\texttt{n\_estimators}${}=500$, lr${}=0.05$,
\texttt{min\_samples\_leaf}${}=5$) and are evaluated on the frozen
expert test set ($n=400$).

\subsection{Key Findings}

\begin{itemize}
    \item \textbf{All three families contribute.} The full 22-feature
          model (0.533 accuracy, 0.489 macro F1) outperforms every
          subset, confirming that the three signal families capture
          complementary information.

    \item \textbf{Opus scores are the strongest single channel.} The
          two-dimensional Opus-only model achieves 0.523 accuracy,
          nearly matching the full model. This demonstrates that a
          well-calibrated LLM-as-judge provides a strong prior that the
          stacker refines rather than replaces.

    \item \textbf{LLM features outperform structural features in
          isolation.} LLM-only (7d) reaches 0.468 accuracy versus 0.393
          for Structural-only (13d), a 7.5 percentage-point gap. However,
          Structural features provide the largest marginal gain when
          combined: LLM+Struct (0.495) exceeds LLM-only by 2.7~pp,
          while LLM+Opus (0.463) is slightly below LLM-only, suggesting
          the Opus signal partially overlaps with the LLM embedding
          signal.

    \item \textbf{Structural features reduce adjacent-accuracy
          degradation.} Structural-only has the lowest adjacent accuracy
          (0.750), but adding structural features to Opus (Struct+Opus:
          0.835) provides better ordinal calibration than LLM+Opus
          (0.818), indicating that graph topology helps the model
          respect the ordinal tier structure.

    \item \textbf{The full model is Pareto-optimal.} No subset achieves
          higher accuracy, F1, or adjacent accuracy than the full model,
          justifying the inclusion of all three feature families despite
          the modest dimensionality overhead.
\end{itemize}

These results confirm that the stacking approach, combining text
similarity, graph structure, and LLM judgment in a single gradient
boosting model, captures relational structure that no single signal
family provides alone.
