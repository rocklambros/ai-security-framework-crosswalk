\section{Ethics Statement}
\label{sec:ethics}

This work aims to assist security practitioners in navigating the
complex landscape of AI security frameworks. The crosswalk system is
intended as a decision-support tool and should not replace expert
judgment. Predicted mappings, especially those not confirmed by the
conformal predictor with small prediction sets, should be reviewed by
qualified subject matter experts before being used in compliance
decisions.

The training data includes publicly available framework texts and
community-contributed crosswalk data released under permissive licenses
(CC~BY-SA~4.0 for upstream GenAI Security Project
data~\citep{genai_security_project}; Apache-2.0 for our code and model
weights). No personally identifiable information is used in any stage
of the pipeline.

The LLM-as-judge component uses a commercial language model for feature
extraction during training. We report this dependency transparently and
note that the resulting features are deterministic given the same model
version and prompts. The expert-labeled test set was produced by a
single annotator; future work should expand to multiple independent
annotators to assess inter-rater reliability and reduce potential bias
in the gold standard.
