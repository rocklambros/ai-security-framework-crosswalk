\section{Method}
\label{sec:method}

Our pipeline comprises two stages: feature extraction and stacking
classification, followed by conformal calibration and uncertainty
routing.

\subsection{Feature Extraction}

For each candidate control pair $(u, v)$, we extract a
50-dimensional feature vector organized into three families:

\paragraph{GAT embeddings (35 dimensions).}
We construct a heterogeneous control graph with nodes from the four
framework sources described in Section~\ref{sec:dataset}. A Graph
Attention Network (GAT) with 32-dimensional node embeddings is trained
on the graph structure, producing learned representations that capture
both local neighborhood topology and multi-hop structural relationships.
For each candidate pair, we extract pairwise features from the GAT
embeddings including element-wise products, absolute differences, and
concatenated summary statistics, yielding 35 structural features.
Figure~\ref{fig:node2vec_umap} visualizes the graph structure via a
UMAP projection of node embeddings, revealing framework-level clustering
with cross-framework bridges.

\begin{figure}[t]
\centering
\includegraphics[width=0.85\linewidth]{fig_node2vec_umap}
\caption{Node embeddings projected to 2D via UMAP, colored by
framework. Clusters correspond to individual frameworks, with

\label{fig:node2vec_umap}
\end{figure}

\paragraph{Cross-encoder probability features (12 dimensions).}
Three pre-trained cross-encoder models are fine-tuned on the
human-calibrated training pairs to predict the four-class tier
distribution:
\begin{enumerate}
    \item DeBERTa-v3-large (304M parameters),
    \item RoBERTa-large (355M parameters),
    \item DeBERTa-v3-base (86M parameters).
\end{enumerate}
Each model produces a 4-class softmax probability vector for every
candidate pair, yielding $3 \times 4 = 12$ probability features. The
architectural diversity (two DeBERTa variants plus RoBERTa) ensures
that the ensemble captures complementary aspects of semantic similarity.
Feature importance analysis shows that RoBERTa features dominate
(occupying the top four positions by coefficient magnitude), followed by
DeBERTa-v3-base, while the original DeBERTa-v3-large features show
lower importance (${\sim}0.02$), suggesting diminishing returns from
model scale within the same architecture family.

\paragraph{Baseline similarity features (3 dimensions).}
Three classical similarity scores complete the feature set:
BGE-large cosine similarity between control text embeddings,
BM25 scores computed over control titles and descriptions, and
a bridge centrality score derived from the graph structure. The bridge
score captures topological proximity independent of text content and
ranks 9th in feature importance, confirming that graph structure
provides complementary signal even in the presence of powerful text
encoders.

\subsection{L2-Regularized Logistic Regression Stacker}

The 50-dimensional feature vector is fed to an L2-regularized
logistic regression classifier that predicts one of four tiers:
\emph{unrelated}~(0), \emph{partial}~(1), \emph{related}~(2), and
\emph{equivalent}~(3). The regularization strength is set to
$C = 0.01$, chosen via 5-fold cross-validation on the 477
human-calibrated training pairs (CV macro F1: 0.861). The strong
regularization prevents overfitting to the cross-encoder probability
features, which would otherwise allow the stacker to memorize
training-set artifacts. Compared to the v6 LightGBM stacker, the
logistic regression model is simpler, more interpretable, and less
prone to overfitting on the modest training set size.

\subsection{Conformal Prediction}

We wrap the logistic regression classifier with a conformal predictor
that applies marginal coverage with label-shift correction. For each
test instance, the conformal predictor outputs a prediction set
guaranteed to contain the true label with probability
$\geq 1-\alpha$, providing distribution-free coverage
guarantees~\citep{vovk2005conformal, romano2020malice}. At
$\alpha = 0.10$, the predictor achieves 91.6\% empirical coverage with
a mean prediction set size of 1.56, a substantial improvement over the
v6 Mondrian conformal predictor (set size 3.20), reflecting the
sharper probability estimates produced by the cross-encoder ensemble.

\subsection{KL-Divergence Router}

A KL-divergence router compares the predicted class distribution against
a uniform reference distribution. Instances with high KL divergence
(confident predictions) are accepted; those with low divergence
(uncertain predictions) are routed to human review. This creates a
natural human-in-the-loop workflow where experts focus on the most
ambiguous cases.
