\section{Results}

\subsection{Main Results}

Table~\ref{tab:main_results} summarizes performance on the 179
expert-labeled frozen test set. The full v7c pipeline (Method~B)
combines GAT embeddings, three cross-encoder probability features, and
baseline similarity scores in an L2-regularized logistic regression
stacker, achieving:

\begin{itemize}
    \item \textbf{Tier accuracy:} 0.810 (95\% CI [0.749, 0.866])
    \item \textbf{Macro F1:} 0.512 (95\% CI [0.437, 0.580])
    \item \textbf{Adjacent accuracy:} 0.944, meaning 94\% of predictions
          fall within one tier of the ground-truth label
    \item \textbf{Binary accuracy:} 0.911
\end{itemize}

\subsection{Per-Class Analysis}

Performance varies substantially across tiers. The Unrelated class
achieves the highest F1 (0.938), benefiting from clear semantic
separation amplified by the cross-encoder probability features. The
Partial and Related classes both reach F1 of 0.556, reflecting the
inherent difficulty of distinguishing these neighboring tiers.

The Equivalent class achieves F1 of 0.000, indicating that the model
fails to identify any Equivalent pairs on the frozen test set. This
reflects the extreme rarity of true equivalences in the test
distribution and the difficulty of distinguishing full semantic
equivalence from strong relatedness. The stacker learns to avoid
predicting the minority Equivalent class when the cost of a
false positive outweighs the benefit.

The dominant error mode has shifted relative to v6. Where the earlier
GBM model suffered from majority-class gravitational pull toward
Related, the v7c ensemble instead achieves strong separation of
Unrelated pairs (F1 0.938) while compressing the middle tiers. This
trade-off is favorable for the primary use case of crosswalk
construction, where confidently identifying unrelated pairs allows
efficient pruning of the candidate space.

\subsection{Ablation Analysis}

We evaluate four method variants to isolate the contribution of each
component family (Table~\ref{tab:main_results}):

\begin{itemize}
    \item \textbf{Method A} (GAT-only, 35 features): accuracy 0.659,
          macro F1 0.367. Graph structure alone provides a useful baseline
          but cannot capture fine-grained semantic distinctions.
    \item \textbf{Method B} (Full pipeline, 50 features): accuracy 0.810,
          macro F1 0.512. The complete feature set yields the best
          performance across all metrics.
    \item \textbf{Method C} (CE-only, 12 CE + 3 baseline features):
          accuracy 0.804, macro F1 0.499. Cross-encoder features alone
          nearly match the full pipeline, confirming that the CE
          probability distributions carry the strongest signal.
    \item \textbf{Method D} (Raw CE average, no stacker): accuracy 0.788,
          macro F1 0.488. Averaging the three cross-encoder predictions
          without a learned stacker performs surprisingly well, indicating
          that the ensemble of diverse architectures provides robust
          probability estimates.
\end{itemize}

The ablation reveals a clear hierarchy: cross-encoder features dominate
(Methods C and D both exceed 78\% accuracy), GAT features alone are
insufficient (Method A at 65.9\%), and the learned stacker provides a
modest but consistent improvement over raw averaging (Method B vs.\
Method D: +2.2 percentage points in accuracy, +0.024 in macro F1).
Feature importance analysis confirms this pattern. RoBERTa-large
features occupy the top four positions by coefficient magnitude,
followed by DeBERTa-v3-base features. The original DeBERTa-v3-large
features show lower importance (${\sim}0.02$), suggesting diminishing
returns from model scale within the same architecture family. The bridge
centrality score ranks 9th, indicating that the graph structure provides
complementary signal even in the presence of powerful text encoders.

\subsection{Conformal Coverage}

The conformal predictor achieves empirical coverage of 91.6\%
at $\alpha = 0.10$, meeting the nominal 90\% target. The mean
prediction set size is 1.56 out of four possible classes, a substantial
improvement over the v6 Mondrian conformal predictor (set size 3.20).
This reduction reflects the stronger discriminative power of the
cross-encoder ensemble: the underlying classifier produces sharper
probability estimates, allowing the conformal wrapper to achieve
coverage with much smaller prediction sets.

\subsection{Expert--LLM Label Gap}

On the LLM-labeled validation set, accuracy rises above the frozen
test baseline, reflecting a distribution gap between expert and LLM
labeling. This gap is most pronounced for the Partial and Equivalent
tiers, where expert annotators apply stricter criteria for semantic
overlap than the LLM-SME labeling protocol. The gap motivates our
emphasis on expert-labeled evaluation and highlights the importance of
the frozen test set for honest benchmarking.
